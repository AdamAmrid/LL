# USN - UM6P Solidarity Network

React + Vite website for the UM6P Solidarity Network (USN).

## Tech Stack
- React + Vite
- React Router
- Tailwind CSS
- Lucide React

## Branding
Colors and typography configured in `tailwind.config.js`.

## Getting Started
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Routes
- `/` Home
- `/about` About
- `/organization` Organization
- `/how-it-works` How It Works

## Accessibility
- Semantic sections and headings
- ARIA labels on nav and interactive elements

## Notes
- Subtle Moroccan geometric background via Tailwind `bg-morocco-pattern`
- Smooth route transitions with a top progress bar
